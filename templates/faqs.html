Find answers to common questions about SpotScrape

SpotScrape is a free tool that extracts metadata from Spotify URLs without requiring API keys. Simply paste any Spotify URL (track, album, playlist, or artist) and get detailed information including titles, descriptions, images, and more.

Some Spotify content may be region-restricted or private. Make sure the URL is public and accessible. If you continue having issues, try refreshing the page or contact support.

History and Settings are only available to registered users. Sign up for a free account to access your extraction history, personalized settings, and additional features.

Your extraction history is saved indefinitely for registered users. You can view, export, or delete individual items anytime from your history page.

User Restrictions:
• Only public Spotify content can be extracted
• Private playlists and region-locked content are not accessible
• Chrome browser required for playlist scraping
• Internet connection required for all operations
• History limited to last 20 items for display

Technical Limitations:
• Large playlists (1000+ tracks) may take longer to process
• YouTube integration requires external API availability
• Some content may be temporarily unavailable due to Spotify changes

SpotScrape is free with no rate limits, but please use responsibly:
• Don't spam requests or overload the server
• Respect Spotify's terms of service
• Use extracted data for personal purposes only
• Don't redistribute copyrighted content
• Report any bugs or issues to help improve the service

Chrome browser is required only for playlist scraping functionality. The app uses Selenium WebDriver with Chrome to extract detailed playlist data. For basic track/album/artist extraction, any modern browser works fine.
